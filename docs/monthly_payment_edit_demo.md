# 月缴费额字段编辑功能演示

## 功能说明

现在用户可以在右侧边栏直接编辑"月缴费额"字段,无需通过对话修改。

## 使用步骤

### 1. 打开应用
在浏览器中访问: http://localhost:5000

### 2. 登录系统
输入您的姓名并登录

### 3. 查看当前信息
在右侧边栏可以看到当前用户信息,包括:
- 基本信息(姓名、性别、出生年月等)
- 社保信息(工作年限、账户余额等)
- 缴费信息(社平工资、缴费基数、**月缴费额**)

### 4. 编辑月缴费额
1. 点击右侧边栏顶部的 **"✏️ 修改信息"** 按钮
2. 在弹出的编辑表单中找到 **"月缴费额"** 字段
3. 修改该字段的值,例如:
   ```
   低档60%:1358元/月,中档100%:2264元/月,高档300%:6793元/月
   ```
4. 点击 **"💾 保存"** 按钮保存修改

### 5. 验证修改
保存后,右侧边栏会自动更新显示新的月缴费额信息。

## 技术实现

### 前端修改 (templates/index.html)

**可编辑字段列表** (第920行):
```javascript
const editableFields = {
    // ... 其他字段 ...
    'flex_payment_bases': '灵活就业缴费基数(100%档,元/月)',
    'flex_monthly_payments': '月缴费额'  // 新增字段
};
```

**数据保存** (第1023行):
```javascript
const updatedData = {
    // ... 其他字段 ...
    flex_payment_bases: document.getElementById('edit_flex_payment_bases')?.value || '',
    flex_monthly_payments: document.getElementById('edit_flex_monthly_payments')?.value || ''  // 新增字段
};
```

### 后端支持

后端 `/api/update-user-data` 接口已经支持 `flex_monthly_payments` 字段的更新,无需修改。

## 注意事项

1. 月缴费额字段是可选字段,可以随时修改
2. 修改后会立即保存到用户数据文件
3. 该字段通常由AI计算后自动填写,手动编辑可以覆盖AI的值
4. 字段值格式建议保持一致性,便于阅读和理解
